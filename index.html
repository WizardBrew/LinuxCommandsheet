🐧 Linux Command Cheat Sheet — Quick & Practical

📁 File & Directory

Command Example 1 Example 2
cd cd /etc/nginx – Go to Nginx config cd .. – Up one level
ls ls -lh – Human-readable sizes ls -a – Show hidden
mkdir mkdir logs – Make logs dir mkdir -p a/b/c – Nested dirs
rm rm file.txt – Remove file rm -r temp/ – Remove folder
cp cp a.txt b.txt – Copy file cp -r src/ dest/ – Copy folder
mv mv old.txt new.txt – Rename mv file.txt /tmp/ – Move file

🌐 Network

Command Example 1 Example 2
ping ping google.com – Test reachability ping -c 4 8.8.8.8 – 4 packets
curl curl https://example.com – Fetch page curl -I https://example.com – Headers only
wget wget file.zip – Download file wget -r https://site.com – Recursive
ip ip a – Show addresses ip r – Show routes
netstat netstat -tuln – Listening ports netstat -p – PID/program
ss ss -tuln – Sockets ss -p – With process
