🐧 Linux Command Cheat Sheet

Quick & Practical

📁 File & Directory Commands

🔧 Command 📌 Example 1 📌 Example 2
cd cd /etc/nginx – Navigate to config folder cd .. – Move up one directory
ls ls -lh – List with human-readable sizes ls -a – Show hidden files
mkdir mkdir logs – Create a folder named logs mkdir -p a/b/c – Create nested folders
rm rm file.txt – Delete file rm -r temp/ – Delete folder recursively
cp cp a.txt b.txt – Copy file cp -r src/ dest/ – Copy folder recursively
mv mv old.txt new.txt – Rename file mv file.txt /tmp/ – Move file to /tmp

🌐 Network Commands

🌍 Command 📌 Example 1 📌 Example 2
ping ping google.com – Check connectivity ping -c 4 8.8.8.8 – Send 4 packets
curl curl https://example.com – Fetch webpage curl -I https://example.com – Get headers only
wget wget file.zip – Download file wget -r https://site.com – Recursive download
ip ip a – Show IP addresses ip r – Show routing table
netstat netstat -tuln – List listening ports netstat -p – Show PID/program name
ss ss -tuln – Show TCP/UDP sockets ss -p – Show process using socket

⚙️ Bash RC & Shell Customization

🧠 Command 📌 Example 1 📌 Example 2
alias alias ll='ls -la' – Create shortcut alias gs='git status' – Git alias
export export PATH=$PATH:/opt/bin – Add to PATH export EDITOR=nano – Set default editor
.bashrc source ~/.bashrc – Reload bash config nano ~/.bashrc – Edit bash config
env env | grep USER – Show user env var env | less – Scroll through env vars
set set -x – Debug shell commands set +x – Turn off debug mode

🔄 Process & System Monitoring

🧪 Command 📌 Example 1 📌 Example 2
ps ps aux – List all processes ps -ef | grep nginx – Search for nginx
top top – Live process monitor top -u parvez – Show Parvez’s processes
htop htop – Interactive process viewer htop -u root – Filter by user
kill kill 1234 – Kill process by PID kill -9 1234 – Force kill process
nice nice -n 10 script.sh – Run with priority renice -n -5 -p 1234 – Change priority of PID
uptime uptime – Show system load uptime -p – Pretty format uptime
